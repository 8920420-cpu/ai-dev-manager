# Миграция 1.2.0 → 1.3.0 (ПРИМЕР формата)

> Пример будущей миграции. Не применяется, пока эталон остаётся на 1.0.0.

- **From:** `1.2.0`
- **To:** `1.3.0`
- **Тип:** пример

## Мотивация (пример)

Перейти на канонический формат этапов `.pipeline.json` `{ "enabled": bool, "commands": [...] }` и
удалить поддержку массива команд.

## Шаги (только каркас)

1. `RENAME` в `.pipeline.json` старый формат массива команд этапа в объект `{ enabled, commands }`.
2. `SET` отсутствующий `enabled` в `true` для существующих этапов.
3. `SET` `.orchestrator/version.json` → `1.3.0`.

## Критерий завершённости

Все этапы `.pipeline.json` в объектном формате, версия структуры = `1.3.0`.
