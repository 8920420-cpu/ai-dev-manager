# Миграция 1.0.0 → 1.1.0 (ПРИМЕР формата)

> Пример будущей миграции. Не применяется, пока эталон остаётся на 1.0.0. Показывает, как описывать
> структурный переход при повышении версии.

- **From:** `1.0.0`
- **To:** `1.1.0`
- **Тип:** пример

## Мотивация (пример)

Добавить стадию верхнеуровневой приоритизации входящих задач.

## Шаги (только каркас)

1. `ADD` папку `tasks/triage/` (`.gitkeep`) между `inbox/` и `ready/`.
2. `ADD` в `tasks/README.md` строку про стадию `triage`.
3. `SET` `.orchestrator/version.json` → `1.1.0`.

## Критерий завершённости

Каркас содержит стадию `triage`, версия структуры = `1.1.0`.
